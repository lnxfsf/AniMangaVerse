API

Route: /api

Method: GET

Description: Route for the API.

List Anime

Route: /api/ListAnime

Method: GET

Description: Route to get a list of anime.

Anime Detail by ID

Route: /api/Anime/<string:pk>/

Method: GET

Description: Route to get details of a specific anime by ID.

Anime Detail by anime_id

Route: /api/Anime/<int:anime_id>/

Method: GET

Description: Route to get details of a specific anime by anime_id.

List Favorites

Method: GET

Description: Route to get a list of favorites.

List Favorites by user_id

Route: /api/ListFavorites/<string:user_id>

Method: GET

Description: Route to get a list of favorites by user ID.

User Registration

Route: /api/v1/register

Method: POST

Description: Route for user registration.

Parameters:

  • username (string, required): Username for registration.
  • email (string, required): Email for registration.
  • password (string, required): Password for registration.

User Login

Route: /api/v1/login

Method: POST

Description: Route for user login.

Parameters:

  • username (string, required): Username for login.
  • password (string, required): Password for login.
