User Registration

Route: /api/v1/register

Method: POST

Description: Route for user registration.

Parameters:

  • username (string, required): Username for registration.
  • email (string, required): Email for registration.
  • password (string, required): Password for registration.

User Login

Route: /api/v1/login

Method: POST

Description: Route for user login.

Parameters:

  • username (string, required): Username for login.
  • password (string, required): Password for login.

List All Users

API Route: /api/ListUsers
Method: GET
Description: Fetches a list of all users in the database.

List One User

API Route: /api/ListUsers/<user_id>
Method: GET
Description: Fetches details of a single user by their user ID.

Path Parameters:

  • user_id (string): The ID of the user to retrieve.

Edit User Profile

API Route: /api/edituserprofile
Methods: GET, POST
Description: Edits the profile of a user. Updates password, username, bio, and profile image if provided.

Request Body (JSON):

  • user_id (string): The ID of the user to update.
  • username (string, optional): The new username.
  • passConfirm (string): The current password.
  • passNew (string, optional): The new password.
  • bio (string, optional): The new bio.
  • profile_image (string, optional): The new profile image URL.
